More ways to get started

Alternative CDNs

  • If want to integrate Modulo into a project that already uses ESM-style import syntax, you can use the Skypack CDN to do so, as follows: 
    import { Modulo, modulo } from 'https://cdn.skypack.dev/mdu.js';
  • You can also pin a specific version of Modulo as such: 
    https://unpkg.com/mdu.js@0.0.8/src/Modulo.js

Download

⌄ Modulo.js (Latest)

Or run the following command:

curl https://unpkg.com/mdu.js -o Modulo.js

NPM (Experimental)

Initialize a Modulo-powered SSG project:

npm init modulo

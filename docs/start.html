Get started with Modulo

Starting snippet (embedded)

Note: This is meant for exploring features. Your work will not be saved.

 
<!-- Include CDN & embed Component -->
<script Modulo src="https://unpkg.com/mdu.js">
    <Component name="BtnEx">
        <Template>
            <button><slot></slot></button>
        </Template>
        <Style>
            button { background: tomato }
        </Style>
    </Component>
</script>
<p><x-BtnEx><em>Hello Modulo</em></x-BtnEx></p>

When keeping your Components in a separate file (e.g. static/components.html), include with the following:

Note: This is meant for exploring features. Your work will not be saved.

 
<script Modulo
    src="https://unpkg.com/mdu.js"
    -src="/static/components.html"
></script>

More ways to get started

Alternative CDNs

  • If want to integrate Modulo into a project that already uses ESM-style import syntax, you can use the Skypack CDN to do so, as follows: 
    import { Modulo, modulo } from 'https://cdn.skypack.dev/mdu.js';
  • You can also pin a specific version of Modulo as such: 
    https://unpkg.com/mdu.js@0.0.16/src/Modulo.js
